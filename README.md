# ProtoCommerce Manual Testing Portfolio

This repository contains my manual QA testing work for the ProtoCommerce demo e-commerce application used for learning and practice purposes.

## Application Under Test
- Name: ProtoCommerce
- Type: Demo e-commerce web application
- URL: https://rahulshettyacademy.com/angularpractice

## Testing Scope
- Shop page product listing
- Product navigation flow
- UI behavior and basic functional testing

## Test Artifacts Included
- Bug reports (Jira and Bugzilla)
- Test scenarios
- Screenshots as evidence

## Sample Bug
One of the identified issues:
- Clicking on a product card redirects the user to an unrelated page instead of the product detail page.

This bug was documented and tracked using:
- Jira (full lifecycle: Open → In Progress → Fixed → Retest → Closed)
- Bugzilla (for comparison and practice)

- Jira
- Bugzilla
