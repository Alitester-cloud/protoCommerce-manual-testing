# Bug ID: ALI-24

## Title
Product click redirects user to Protractor Tutorial page instead of product details

## Application
ProtoCommerce demo e-commerce application

## Environment
- OS: Windows 10
- Browser: Chrome (latest)
- URL: https://rahulshettyacademy.com/angularpractice

## Preconditions
- User is on the ProtoCommerce Shop page
- Products are visible on the screen

## Steps to Reproduce
1. Open the ProtoCommerce Shop page
2. Click on any product card (e.g. iPhone X)

## Expected Result
User should be navigated to the selected product detail page displaying relevant product information.

## Actual Result
User is redirected to an unrelated “Protractor Tutorial” form page.

## Severity
Medium

## Priority
High

## Tools Used
- Jira
- Microsoft Excel

## Notes
This bug was tracked through the full Jira lifecycle: Open → In Progress → Fixed → Retest → Closed.
